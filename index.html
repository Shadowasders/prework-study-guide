Prework Study Guide

HTML

  • The head element contians information about the meta-data of the webpage
  • The body element contains all the elements on the webpage that users will be able to see.
  • Use Href with A elements, and src with Image elements.
  • When setting up the viewport and width of a page, always use name="viewport" and content="width=device-width, initial-scale=1.0"

CSS

  • A margin indicates how much space we want outisde of an element
  • Padding indicates how much space we want around the content INSIDE an element
  • CSS selectors can be used to select more than one element at once using a comma between them
  • All HTML elements are set as boxes on a webpage, and set as block by default, by setting elements as inline, one could set more than one element per line
  • Colors in CSS can be set semantically, using a hexcode, or by using rgb.

Git

  • Use li to check which files are around you when exploring in bash.
  • Use git checkout -b branch name to create a new branch and move to it.
  • Use git checkout branch name to move to other branches
  • One can autofill names of repos using tab
  • Use Cd to change directories, and use cd .. to go back
